# personal_budget_tracker (excel)
## overview
this project is a personal budget tracker created using MICROSOFT EXCEL.
it helps track income, expenses, savings, and provide a summary dashboard.

## sheets included
1. summary (dashboard)
2. income
3. expenses
4. savings

## features
- automated calculations using SUM AND PERCENTAGE formulas
- interactive dashboard
- all sheets are interlinked
- easy to customize for personal use

  ## tools used
  - MICROSOFT EXCEL
